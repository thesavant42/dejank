# dejank

by thesavant42

## dejank uglified apps

### Install

Install with:

`go install github.com/thesavant42/dejank/cmd/dejank@latest`

### Run It

Run like this:

```bash
dejank.exe url http://example.app/

[*] Target: http://example.app/

[*] Summary
  Scripts discovered: 46
  Maps discovered:   41
  Sources restored:  1524
  Assets extracted:  0
  ```

Output will be in `./example.app/` or wherever you specify.
